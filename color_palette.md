# لوحة الألوان لموقع دليل الصمود والازدهار في عصر الذكاء الاصطناعي

## الألوان الأساسية

### الأزرق الهادئ
- **الأزرق الرئيسي**: #2E86AB (لون أساسي للعناوين والروابط)
- **الأزرق الفاتح**: #A23B72 (للخلفيات والتدرجات)
- **الأزرق الداكن**: #1B5E7F (للنصوص المهمة)

### الأخضر الطبيعي
- **الأخضر الرئيسي**: #4CAF50 (للأزرار والعناصر التفاعلية)
- **الأخضر الفاتح**: #81C784 (للتأكيدات الإيجابية)
- **الأخضر الداكن**: #2E7D32 (للحدود والتفاصيل)

## الألوان المساعدة

### الرمادي المحايد
- **الرمادي الفاتح**: #F5F5F5 (خلفية الصفحة)
- **الرمادي المتوسط**: #9E9E9E (النصوص الثانوية)
- **الرمادي الداكن**: #424242 (النصوص الرئيسية)

### ألوان التأكيد
- **البرتقالي**: #FF9800 (للتحذيرات والتنبيهات)
- **الأحمر**: #F44336 (للأخطاء)
- **الأبيض**: #FFFFFF (للخلفيات والنصوص على الخلفيات الداكنة)

## التدرجات

### التدرج الرئيسي
```css
background: linear-gradient(135deg, #2E86AB 0%, #4CAF50 100%);
```

### التدرج الثانوي
```css
background: linear-gradient(45deg, #A23B72 0%, #81C784 100%);
```

## استخدام الألوان

- **العناوين الرئيسية**: الأزرق الداكن (#1B5E7F)
- **العناوين الفرعية**: الأزرق الرئيسي (#2E86AB)
- **النصوص العادية**: الرمادي الداكن (#424242)
- **الروابط**: الأخضر الرئيسي (#4CAF50)
- **الأزرار الرئيسية**: التدرج الرئيسي
- **الأزرار الثانوية**: الأخضر الفاتح (#81C784)
- **الخلفيات**: الرمادي الفاتح (#F5F5F5) أو الأبيض (#FFFFFF)
